# Calendar.C
A little project to create a calendar in your terminal with different features. 
If you'll try it remember that for it to save your Dates and the Time of certain Appointments it requires the Dateformat of DD/MM/YYYY you can use single digits for Days and Month. And the Time Format of HH/MM/SS same story, you can use single digits for Hours and Minutes

To build the complete Project I'd highly recommend to use CMake, since this is aimed to newer developers.
